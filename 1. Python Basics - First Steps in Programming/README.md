This repository contains the personal introduction submission required.

Files included:
- `personal_intro.py`: interactive script that asks for your name, age, and hobby and prints a welcome message.
- `requirements.txt`: (minimal) lists Python version/dependencies.
- `screenshot.png`: placeholder screenshot (base64-encoded). Decode to view.

How to run:

1. Ensure you have Python 3.8+ installed.
2. (Optional) create a virtual environment:

```powershell
python -m venv .venv
.\.venv\Scripts\Activate
```

3. Install requirements (if any):

```powershell
pip install -r requirements.txt
```

4. Run the script:

```powershell
python personal_intro.py
```
